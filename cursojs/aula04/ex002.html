no javascript moderno, além de utilizar a palavra var para variáveis, também podemos uar a palavra let.

existem 3 formar de delimitar uma string em javascript: "", '', ``, cada uma é utilizada de forma diferente.

indicadores podem começar com letra, $ ou _; não podem começar com números; é possivel usar letras e números; é possível usar acentos e símbolos; não podem contar espaços; não podem ser palavras reservadas.

Dicas: maiúsculas e minpusculas fazem diferença(são case sensitive); tente escolher nomes coerentes para as variáveis;

Data Types (tipos primitivos) [typeof]: number (inteiros e flutuantes) [Infinity, NaN], string (cadeia de caracteres), boolean (true, false), [null, undefined, object (Array), function].
